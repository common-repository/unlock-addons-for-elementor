Unlock Addons for Elementor offers 6 powerful widgets to enhance your website building experience using Elementor.

== Description ==

**Unlock Addons for Elementor** offers a variety of custom-designed Elementor widgets that enable you to create stunning web pages without needing any coding skills. The plugin adds 6 unique widgets, which help you quickly add dynamic elements to your pages. From grid layouts to stylish buttons, Unlock Addons brings creative freedom to your Elementor editor, and it’s totally free.

**Available Widgets:**
1. Blog Grid
2. Team Grid
3. Button
4. Progress Bar
5. Woocommerce Product Carousel
6. Woocommerce Product Grid


Each widget is fully customizable, allowing you to design according to your brand’s needs.

**SEO Optimized:**
Unlock Addons for Elementor is developed with SEO in mind, ensuring faster load times and smooth user experience. By using the widgets, your site’s performance won’t be compromised, making it easier to rank higher in search engine results.

**PRO Features:**
At this time, there is no pro version of the plugin available.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/unlock-addons-for-elementor` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Edit a page with Elementor, then search for the Unlock Addon category.
4. Start using the available widgets to enhance your page designs!

== Frequently Asked Questions ==

= How do I use Unlock Addons widgets? =

Simply install and activate the Unlock Addons plugin. Then, while editing any page with Elementor, search for the Unlock Addon widgets in the widget panel.

= Elementor editor is not loading or working properly? =

This issue is likely related to your server's PHP settings. You can fix this by increasing your PHP memory limit from the `wp-config.php` file or `php.ini` file. If you are unsure how to do this, contact your hosting provider and ask them to increase the following settings:
– PHP `memory_limit` = 512M
– `max_execution_time` = 300
